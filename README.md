# Vertila — Magic Draw (MVP)

MVP веб-продукта для фокуса с предсказанием: зритель рисует на `/draw`, а скрытая логика на датчиках движения фиксирует один из исходов и выбирает предсказание из настроек `/admin`.

## Запуск

```bash
npm install
npm run dev -- --host
```

Откройте:
- `http://localhost:5173/draw` — spectator page (телефон зрителя)
- `http://localhost:5173/admin` — admin panel (настройка)

Чтобы протестировать с телефона в одной сети:
1) запустите `npm run dev -- --host`
2) откройте IP компьютера: `http://<LAN-IP>:5173/draw`

## Где расширять до полноценного 8-mode

Сейчас надежная классификация реализована для 4 направлений. Для режима `mode=8` данные/конфиг уже поддерживают 8 предсказаний и структуру `mapping8`, но классификатор пока делает **graceful fallback** на 4-направления (это видно в debug-режиме).

Точка расширения:
- `src/hooks/useMotionClassifier.ts` — место, где после вычисления 4-направления можно заменить/расширить классификацию до 8 исходов (и использовать `config.mapping8`).
- `src/types/config.ts` — структура `Mapping8` (placeholder-ready).
